Amur Baliqchi — статический сайт
================================

Структура:
- index.html — главная
- about.html — о компании
- products.html — продукция
- services.html — услуги
- contact.html — контакты
- assets/css/styles.css — стили
- assets/js/main.js — JS (обработчик формы-заглушки)
- assets/img/logo.svg, hero_wave.svg — базовые изображения
- assets/icons/favicon.png, favicon-32.png — фавиконки

Как использовать:
1) Разверните содержимое на любом хостинге (nginx/Apache, Netlify, Vercel, GitHub Pages).
2) В contact.html подключите реальную обработку формы (email/telegram/bot/serverless).
3) Отредактируйте телефоны, email, адрес и юридические реквизиты.
4) Замените изображения на реальные фото хозяйства.

Лицензия: файлы предназначены как стартовый шаблон и могут свободно изменяться.
